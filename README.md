# Orbitarium

*A celestial sandbox for exploring order, chaos, and gravity.*

---

## Overview

Orbitarium is an interactive 3D N-body simulation designed not only as a physics experiment, but also as a space observation platform.

Watch planetary systems emerge, stabilize, collapse, and scatter under the influence of gravity. Explore orbital mechanics through a mission-control-inspired interface featuring telemetry, barycenter tracking, angular momentum visualization, orbital cameras, and curated simulation scenarios.

What began as a simple gravity simulator gradually evolved into a personal mission control center for observing miniature universes.

---

## Features

### Physics Engine

* N-body gravitational simulation
* Dynamic orbital systems
* Stable and unstable configurations
* Chaotic gravitational interactions
* Adjustable simulation speed

### Observation Tools

* OrbitCam cinematic tracking
* Barycenter visualization
* Angular momentum monitor
* Planet trails
* Comet trails
* Solar trajectory tracking
* Multiple camera presets

### Presets

#### Stable Orbit

A balanced planetary system demonstrating long-term orbital stability.

#### Chaos Collapse

A fragile system that gradually destabilizes and collapses under gravitational interactions.

#### Scatter Test

A close-encounter scenario showcasing slingshot effects and orbital scattering.

#### Render Load

A stress test for exploring rendering and simulation performance.

### Mission Control Interface

* Retro NASA-inspired console
* Real-time telemetry
* Guidance monitor
* Save / Load support
* Keyboard shortcuts
* English / Japanese localization

## Philosophy

Orbitarium is not intended to be a professional astrophysics simulator.

Its purpose is to make gravity visible.

Some systems settle into elegant harmony.

Others descend into chaos.

Both outcomes are worth observing.
